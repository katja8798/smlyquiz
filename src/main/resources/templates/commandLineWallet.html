Setting up a command-line Smileycoin wallet (Linux) on Mac OS

To be able to use the command-line wallet one must have a Linux system so in this chapter we will be going through the steps of installing the wallet on a Mac OS with the help of Virtual Box. There are multiple ways of setting up a command-line wallet for a mac, this just demonstrates one way of doing it.

Install Virtual Box and Ubuntu

  1. Install virtual box from here
  2. Install Ubuntu LTS version from here
  3. Set up your virtual machine and ubuntu
    4. For detailed instructions check out this website

Set up wallet in a container

Once everything is installed and you have your machine ready, open the terminal on your virtual machine and do the following steps:

      Step One
  1. sudo apt-get install lxd
  2. sudo lxd init
  3. sudo lxc launch ubuntu:20.04 containerName
    4. containerName will be the name of the lxd container.
      Step Two - as a root in the container
  1. sudo lxc exec containerName -- bin/bash
  2. apt update
  3. apt-get upgrade
  4. apt install debhelper libssl-dev openssh-client openssh-server build-essential pkg-config libtool autotools-dev autoconf libboost-all-dev libdb-dev libdb++-dev
  5. adduser myUser
      Step Three - as the user in the container
  1. lxc exec containerName -- su myUser
  2. git clone https://github.com/smileycoin/smileyCoin
  3. cd smileyCoin
  4. ./autogen.sh
  5. ./configure
  6. make

Then the wallet should be ready in your container!
