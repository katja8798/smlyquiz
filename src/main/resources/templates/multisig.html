How to set up a multisig address

Multisig allows users to have one smileycoin address with multiple key holders, so funds from the address cannot be spent until all key holders sign off on it. In order to create a multisig address, you will need two or more public public keys to generate it.

We have two methods for generating the address. One is the command createmultisig. Example: createmultisig 3 ‘[“publickey1", "publickey2", "publickey3”]'

The other is the addmultisigaddress command: addmultisigaddress 3 ‘[“publickey1", "publickey2", "publickey3”]'

The output from the command is the “multisig address”. After you´ve sent some amount to the multisig address, you can send transaction from it. All key holders will need to sign the transaction.
